## 🚀 How to View Formatted Guides
Since the formatted guides use **Markdown**, standard text editors like Notepad might show the raw text layout. To view them rendered with styling (tables, headings, code highlights):
1. **VS Code**: Open any `- Formatted.md` file and press `Ctrl + Shift + V` to launch the live graphical preview.
2. **GitHub/GitLab**: When browsed on host systems, these files will render automatically.
3. **Web**: You can drop any `.md` file into web editors like [StackEdit](https://stackedit.io/) to render them on any browser.
